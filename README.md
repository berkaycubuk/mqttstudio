# Basic E-Com

!!!IN DEVELOPMENT!!!

This is a minimal e-commerce project written in Go. Main goal is to be small, and simple.

## Tech

- Go
- Sqlite3
- HTML
- CSS
- JavaScript

## Features

- Product listing
- Product details
- Cart system
- Admin controls like adding a new product

## Development

1) `go get` to install required libraries
2) `go run .` to run the project
